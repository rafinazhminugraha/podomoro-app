# Audio Files Guide

Place the following audio files in this directory (`/public/audio/`):

## Required Files

| File Name | Purpose | Duration | Recommended Type |
|-----------|---------|----------|------------------|
| `focus-music.mp3` | Background music during focus sessions | 30+ minutes | ADHD-friendly beats, binaural beats, lo-fi |
| `break-music.mp3` | Background music during breaks | 10+ minutes | Calm lo-fi, ambient, nature sounds |
| `alarm-start.mp3` | Sound when focus session starts | 2-5 seconds | Gentle chime, soft bell |
| `alarm-break.mp3` | Sound when break begins | 2-5 seconds | Relaxing tone, soft notification |

## Recommended Free Resources

- [Lofi Hip Hop Coffee](https://pixabay.com/music/search/lofi/) - Pixabay

- [Relaxing Piano](https://pixabay.com/music/search/piano%20relaxing/)

### Alarm Sounds (Gentle)
- [Notification sounds](https://pixabay.com/sound-effects/search/notification/)
- [Bell chimes](https://freesound.org/search/?q=bell+chime)
- [Soft tones](https://www.zapsplat.com/sound-effect-categories/ui-tones/)

## Volume Guidelines
- Music files should be normalized to avoid sudden volume jumps
- Alarm sounds should be clear but not jarring
- The app sets music volume to 30% and alarm volume to 50%

## File Format
- Use MP3 format for best browser compatibility
- 128-256 kbps quality is sufficient
- Ensure files loop smoothly (for music tracks)

## After Adding Files
No code changes needed! The app will automatically use these files.
